DeskMirror

将主屏 Explorer 桌面上的图标镜像到副屏，按比例映射位置，在透明 Overlay 上完成双击打开、选中高亮、拖放与常用右键操作。

GitHub 仓库 下载发布版

主要能力

  • 读取主屏桌面图标（名称、位置），通过 SysListView32 跨进程同步坐标。
  • 副屏无边框透明 Overlay，贴桌面底层 Z-order，适配 PerMonitorV2 DPI。
  • 双击行为贴近真实桌面：已有窗口则激活、托盘应用可尝试 UI Automation / 工具栏唤出，否则 ShellExecute 启动。
  • 支持拖出到副屏其它图标、从资源管理器拖入打开或复制到文件夹；桌面变更与坐标轮询保持同步。
  • 系统托盘：显示/隐藏、刷新、设置（显示器、图标大小、轮询间隔、镜像布局、开机自启）。

运行环境

  • 操作系统：Windows 10 / 11（64 位）。
  • 技术栈：C# / WPF / .NET 10，Release 构建可为单文件自包含，一般无需单独安装运行时。
